This model has low accuracy, this is to be expected since Naive Bayes models make strong assumptions about the dependence of the variables, and we saw in the previous analysis that most of the variables aren't independent. We could probably try to select a few variables that are independent, but it would be best to move on to other models.

Bayesian Network¶

Since this model doesn't make strong assumptions about the independence of the variables and tries to model the relationship between variables, it should give us better results than the Naive Bayes model.

In [18]:
np_train_dataset = train_dataset.drop(["MonthlyCharges", "TotalCharges", "tenure"], axis="columns").to_numpy()
np_test_dataset = test_dataset.drop(["MonthlyCharges", "TotalCharges", "tenure"], axis="columns").to_numpy()

to_predict = test_dataset.drop(["MonthlyCharges", "TotalCharges", "tenure"], axis="columns").assign(Churn = None).to_numpy()

bn = BayesianNetwork.from_samples(np_train_dataset, algorithm='chow-liu', 
                                    state_names=train_dataset
                                        .drop(["MonthlyCharges", "TotalCharges", "tenure"], axis="columns")
                                        .columns.values)
